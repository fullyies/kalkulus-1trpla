    • 6.1 Nilai Maksimum & Minimum

    Definisi
    Nilai maksimum dan minimum suatu fungsi adalah nilai tertinggi dan terendah yang dicapai oleh fungsi tersebut pada interval tertentu.
    Titik Kritis

    Titik Kritis adalah titik pada suatu fungsi dimana turunan pertama bernilai nol atau tidak terdefinisi.
    Secara matematis :
    - Jika f'(x) = 0, atau
    - f'(x) tidak terdefinisi
    maka x disebut titik kritis
    Tujuan : Menentukan titik maksimum dan minimum dari suatu fungsi dengan menggunakan turunan pertama.

    Uji Turunan Pertama

    Uji turunan pertama adalah metode untuk menentukan apakah suatu titik kritis merupakan titik maksimum, minimum, atau bukan keduanya dengan melihat perubahan tanda turunan pertama.
    Aturan :
    - Jika f'(x) Positif -> Negatif = titik maksimum.
    - Jika f'(x) Negatif -> Positif = titik minimum.
    - Jika f'(x) tidak berubah tanda, maka titik tersebut bukan titik ekstrem.

    Uji Turunan Kedua

    Uji turunan kedua adalah metode untuk menentukan jenis titik kritis menggunakan turunan kedua
    Aturan :
    - Jika f"(x) > 0 titik minimum (grafik cekung ke atas).
    - Jika f"(x) < 0 titik maksimum (grafik cekung ke bawah).
    - Jika f"(x) = 0 tidak dapat disimpulkan, sehingga perlu menggunakan uji turunan pertama atau metode lain.

    Nilai Ekstrem

    Nilai ekstrem adalah nilai terbesar (maksimum) atau nilai terkecil (minimum) yang dicapai oleh suatu fungsi
    Jenis - jenis nilai ekstrem :
    - Maksimum lokal : Nilai terbesar dibandingkan nilai fungsi di sekitar titik tersebut.
    - Minimum lokal : Nilai terkecil dibandingkan nilai fungsi di sekitar titik tersebut.
    - Maksimum global (absolut) : Nilai terbesar pada seluruh domain fungsi.

    - Minimum global (absolut) : Nilai terkecil pada seluruh domain fungsi.

    6.2 Contoh Soal

    Contoh Soal 6.1 — Titik Kritis

    Soal:

    Tentukan titik krtis dari fungsi
    f(x)= x^2 −4x + 3

    Penyelesaian:

    Turunan pertama
    f'(x) = 2x - 4

    Tentukan titik kritis
    Karena titik kritis terjadi saat
    f'(x) = 0
    maka
    2x - 4 = 0
    2x = 4 -> 2
    x = 2

    Contoh Soal 6.2 — Uji turunan pertama

    Soal:

    Tentukan apakah fungsi berikut memiliki titik maksimal atau minimum menggunakan uji turunan pertama
    f(x)= x^2 − 4x + 3

    Penyelesaian:

    1. Hitung turunan pertama
    f'(x) = 2x - 4
    2. Tentukan titik kritis
    2x - 4 = 0
    x = 2
    3. Uji tanda turunan pertama
    Ambil satu nilai di kiri dan kanan titik kritis
    - untuk x = 1
    f'(1) = 2(1) - 4 = -2
    - untuk x = 3
    f'(3) = 2(3) - 4 = 2
    Karena nilai berubah dari negatif ke positif, maka fungsi memiliki nilai Minimum pada
    x = 2

    Contoh Soal 6.2 — Uji turunan kedua

    Soal:

    Tentukan apakah fungsi berikut memiliki titik maksimal atau minimum menggunakan uji turunan kedua
    f(x)= -x^2 + 6x - 5

    Penyelesaian:

    1. Hitung turunan pertama
    f'(x) = -2x + 6
    2. Tentukan titik kritis
    -2x + 4 = 0
    x = 3
    3. Uji tanda turunan kedua
    f"(x) = -2
    Subtitusi
    f"(3) = -2
    karena
    -2 < 0
    Maka fungsi memiliki nilai Maksimum pada
    x = 3

    Contoh Soal 6.2 — Nilai Ekstrem

    Soal:

    Tentukan nilai ekstrem dari
    f(x)= x^2 − 4x + 3

    Penyelesaian:

    1. Hitung turunan pertama
    f'(x) = 2x - 4
    2. Tentukan titik kritis
    2x - 4 = 0
    x = 2
    3. Hitung turunan kedua
    f"(x) = 2
    Karena f"(2) = 2 > 0, maka fungsi memiliki nilai Minimum
    4. Hitung nilai extrem
    f(2) = 2^2 - 4(2) + 3
    = 4 - 8 + 3
    = -1

    Jawaban:

    - Titik kritis: x = 2
    - Jenis nilai: Minimum
    - Nilai ekstrem: -1

    Latihan Soal

    Kelompok 6 — Isi latihan soal di bawah ini.

    1. Soal 1

      Diketahui fungsi
      f(x) = x^2 - 6x + 5
      Tentukan :
      1. Titik kritis
      2. Uji turunan pertama
      3. Uji Turunan kedua
      4. Nilai ektrem

      Lihat Jawaban
      1. Menentukan Titik Kritis
      Turunan pertama:
      f'(x) 2x - 6

      Titik Kritis: 2x - 6 = 0
      x = 3 -> Titik kritis

      2. Uji Turunan Pertama
      Ambil nilai di kiri dan kanan titik kritis
      Untuk x = 2
      f'(2) = 2(2) - 6 = -2
      Negatif(-)
      Untuk x = 4
      f'(4) = 2(4) - 6 = 2
      Positif(+)
      Perubahan tanda dari negatif -> positif jadi nilainya adalah minimum

      3. Uji Turunan Kedua
      Turunan kedua
      f"(x) = 2
      Karena
      f"(3) = 2>0
      Jadi nilainya adalah minimum

      Nilai Ekstrem
      f(3) = 3^2 - 6(3) + 5
      = 9 - 18 + 5
      = -4

      Jawaban Akhir
      - Titik kritis = x = 3
      - Uji Turunan Pertama = Minimum
      - Uji Turunan Kedua = Minimum
      - Nilai Ekstrem = -4

    2. Soal 2

      Diketahui fungsi
      f(x) = -x^2 + 8x - 7
      Tentukan :
      1. Titik kritis
      2. Uji turunan pertama
      3. Uji Turunan kedua
      4. Nilai ektrem

      Lihat Jawaban
      1. Menentukan Titik Kritis
      Turunan pertama:
      f'(x) -2x + 8

      Titik Kritis: -2x + 8 = 4
      x = 4 -> Titik kritis

      2. Uji Turunan Pertama
      Ambil nilai di kiri dan kanan titik kritis
      Untuk x = 3
      f'(3) = -2(3) + 8 = 2
      Positif(+)
      Untuk x = 5
      f'(5) = -2(5) + 8 = -2
      Negatif(-)
      Perubahan tanda dari positif -> negatif jadi nilainya adalah maksmimum

      3. Uji Turunan Kedua
      Turunan kedua
      f"(x) = -2
      Karena
      f"(-2) = -2<0
      Jadi nilainya adalah maksimum

      Nilai Ekstrem
      f(4) = -4^2 + 8(4) - 7
      = -16 + 32 - 7
      = 9

      Jawaban Akhir
      - Titik kritis = x = 4
      - Uji Turunan Pertama = Maksimum
      - Uji Turunan Kedua = Maksimum
      - Nilai Ekstrem = 9
