1.1 Konsep Fungsi

Konsep Fungsi adalah aturan yang memasangkan setiap elemen pada himpunan asal (domain) dengan tepat satu elemen pada himpunan tujuan (kodomain). Hasil pemetaan yang benar-benar diperoleh disebut range (daerah hasil).

Dalam matematika, fungsi merupakan suatu hubungan yang menyatakan bahwa setiap nilai input (x) hanya mempunyai satu pasangan nilai output (y). Konsep ini menjadi dasar dalam pembelajaran kalkulus karena digunakan untuk memodelkan hubungan antar variabel.

Domain

Domain adalah himpunan asal (A), yaitu seluruh nilai yang boleh dimasukkan ke dalam suatu fungsi.

Contoh

Jika f(x)=2x+1 dan x merupakan bilangan real, maka domainnya adalah semua bilangan real (ℝ).

Kodomain

Kodomain adalah himpunan tujuan (B) yang menjadi target pemetaan dari suatu fungsi.

Semua hasil fungsi akan berada di dalam kodomain, meskipun tidak semua anggotanya harus dihasilkan.

Range

Range (daerah hasil) adalah himpunan nilai output yang benar-benar dihasilkan oleh fungsi.

Range ⊆ Kodomain
Artinya, semua anggota range pasti berada di dalam kodomain, tetapi belum tentu semua anggota kodomain menjadi range.
Ringkasan Konsep
Istilah Pengertian
Domain Himpunan semua nilai input yang diperbolehkan.
Kodomain Himpunan tujuan dari suatu fungsi.
Range Himpunan nilai output yang benar-benar dihasilkan oleh fungsi.

1.2 Contoh Domain, Kodomain, dan Range

Berikut merupakan beberapa contoh penentuan domain, kodomain, dan range pada berbagai jenis fungsi.
Contoh 1 — Fungsi Linear
f(x)=4x+2

Tentukan domain, kodomain, dan range dari fungsi di atas.

  • Domain : {x | x ∈ ℝ}
  • Kodomain : {y | y ∈ ℝ}
  • Range : {y | y ∈ ℝ}
Karena fungsi linear tidak memiliki batas nilai, maka domain, kodomain, dan range adalah semua bilangan real.
Contoh 2 — Fungsi Kuadrat
f(x)=x²−6x+7

Tentukan domain, kodomain, dan range.

  • Domain : {x | x ∈ ℝ}
  • Kodomain : {y | y ∈ ℝ}
  • Range : y ≥ -2
Fungsi kuadrat mempunyai nilai minimum -2, sehingga range dimulai dari -2 hingga tak hingga.
Contoh 3 — Fungsi Akar
f(x)=√(2x−4)

Karena berbentuk akar, maka syaratnya:

2x−4 ≥ 0
  • Domain : x ≥ 2
  • Kodomain : ℝ
  • Range : y ≥ 0
Contoh 4 — Fungsi Pecahan

Pada fungsi pecahan, penyebut tidak boleh bernilai nol.

  • Domain diperoleh dengan menentukan penyebut ≠ 0.
  • Kodomain adalah semua bilangan real.
  • Range diperoleh menggunakan invers fungsi sesuai syarat yang berlaku.
Ingat! Penyebut fungsi pecahan tidak boleh sama dengan nol.

1.3 Representasi Fungsi

Suatu fungsi dapat direpresentasikan dalam beberapa bentuk, yaitu menggunakan rumus, tabel, dan grafik. Ketiga bentuk tersebut memiliki makna yang sama, hanya berbeda dalam cara penyajiannya.
Representasi Penjelasan Contoh
Rumus Fungsi dinyatakan dalam bentuk persamaan matematika. f(x)=3x−1
Tabel Menampilkan pasangan nilai input (x) dan output (y). (x,y) = (0,-1), (1,2), (2,5)
Grafik Hubungan antara x dan y digambarkan pada bidang Kartesius. Grafik garis lurus.
Contoh Representasi Fungsi Linear

Diketahui fungsi f(x)=3x−1 dengan domain -2 ≤ x ≤ 5.

x -2 -1 0 1 2 3 4 5
f(x) -7 -4 -1 2 5 8 11 14
Menentukan Titik Potong Grafik
  • Titik potong sumbu-Y: x = 0 → y = -1 sehingga diperoleh titik (0,-1).
  • Titik potong sumbu-X: y = 0 → 3x−1 = 0 → x = 1/3 sehingga diperoleh titik (1/3,0).
Setelah kedua titik diperoleh, hubungkan keduanya sehingga terbentuk grafik garis lurus.

1.4 Fungsi Linear dan Nonlinear

Fungsi dapat dibedakan menjadi fungsi linear dan fungsi nonlinear berdasarkan bentuk persamaan serta grafik yang dihasilkan.
Fungsi Linear
f(x)=ax+b

Fungsi linear adalah fungsi yang memiliki pangkat tertinggi satu sehingga menghasilkan grafik berupa garis lurus.

Ciri-ciri:
  • Grafik berupa garis lurus.
  • Gradien bernilai tetap.
  • Laju perubahan konstan.
  • Memiliki bentuk umum ax+b.
Contoh: f(x)=2x+3
Fungsi Nonlinear
f(x)=x²+2x+1

Fungsi nonlinear adalah fungsi yang grafiknya tidak berbentuk garis lurus karena memiliki pangkat lebih dari satu atau bentuk fungsi lainnya.

Ciri-ciri:
  • Grafik berupa kurva.
  • Gradien berubah-ubah.
  • Laju perubahan tidak konstan.
  • Contohnya fungsi kuadrat, akar, eksponen, dan logaritma.
Contoh: f(x)=x²+2x+1
Perbedaan Fungsi Linear dan Nonlinear
Aspek Linear Nonlinear
Bentuk Umum ax + b Selain ax + b
Grafik Garis lurus Kurva
Gradien Tetap Berubah-ubah
Laju Perubahan Konstan Tidak konstan
Contoh f(x)=2x+3 f(x)=x²+2x+1
Kesimpulan: Fungsi linear mempunyai grafik berupa garis lurus dengan gradien tetap, sedangkan fungsi nonlinear menghasilkan grafik berupa kurva karena laju perubahannya tidak konstan.

1.5 Komposisi Fungsi

Komposisi fungsi adalah operasi yang menggabungkan dua fungsi sehingga menghasilkan fungsi baru. Hasil suatu fungsi menjadi input bagi fungsi lainnya.
Notasi Komposisi Fungsi

Jika terdapat dua fungsi f(x) dan g(x), maka komposisi fungsi ditulis sebagai:

(f ∘ g)(x) = f(g(x))

Artinya, nilai x terlebih dahulu dimasukkan ke fungsi g, kemudian hasilnya digunakan sebagai input fungsi f.

Langkah Menentukan Komposisi Fungsi
  1. Tentukan fungsi yang dikerjakan terlebih dahulu.
  2. Substitusikan hasil fungsi pertama ke fungsi berikutnya.
  3. Sederhanakan bentuk aljabarnya.
Contoh Perhitungan

Diketahui:

  • f(x)=2x+3
  • g(x)=x²

Menentukan (f ∘ g)(x)

(f∘g)(x)=f(g(x)) =2(x²)+3 =2x²+3

Menentukan (g ∘ f)(x)

(g∘f)(x)=g(f(x)) =(2x+3)² =4x²+12x+9
Kesimpulan: Komposisi fungsi (f∘g)(x) belum tentu sama dengan (g∘f)(x). Oleh karena itu, urutan penyusunan fungsi sangat memengaruhi hasil akhir.

1.6 Perubahan Input

Perubahan input adalah perubahan nilai variabel bebas (x) yang menyebabkan perubahan pada nilai fungsi f(x). Besarnya perubahan input biasanya dinyatakan dengan simbol Δx (delta x).
Konsep Perubahan Input

Jika nilai input berubah dari x menjadi x + Δx, maka nilai fungsi juga berubah dari f(x) menjadi f(x + Δx).

Δx = xbaru - xlama Δf = f(x + Δx) - f(x)
Contoh Perubahan Input

Diketahui fungsi:

f(x)=2x+1

Jika nilai x berubah dari 2 menjadi 3, maka:

x f(x)
2 5
3 7
  • Δx = 3 - 2 = 1
  • Δf = 7 - 5 = 2
Kesimpulan: Setiap perubahan nilai input akan menghasilkan perubahan pada nilai output. Besarnya perubahan output bergantung pada bentuk fungsi yang digunakan.

1.7 Laju Perubahan

Laju perubahan menunjukkan seberapa besar perubahan nilai output terhadap perubahan nilai input. Dalam fungsi, laju perubahan rata-rata dihitung dengan membandingkan perubahan output dengan perubahan input.
Rumus Laju Perubahan
Laju Perubahan = Δf / Δx

atau dapat dituliskan sebagai:

(f(x₂) − f(x₁)) / (x₂ − x₁)
Contoh Perhitungan

Diketahui fungsi:

f(x)=2x+1

Tentukan laju perubahan pada interval x=2 hingga x=5.

x f(x)
2 5
5 11

Penyelesaian:

Δf = 11 − 5 = 6 Δx = 5 − 2 = 3 Laju Perubahan = 6 / 3 = 2
Interpretasi

Nilai laju perubahan sebesar 2 berarti setiap kenaikan 1 satuan pada nilai input akan menyebabkan nilai output bertambah 2 satuan.

Kesimpulan: Pada fungsi linear, laju perubahan selalu tetap karena gradiennya konstan. Nilai inilah yang menentukan kemiringan grafik garis lurus.

1.8 Sensitivitas Fungsi Linear dan Kuadrat

Sensitivitas fungsi menunjukkan seberapa besar perubahan nilai output akibat perubahan nilai input. Setiap jenis fungsi memiliki tingkat sensitivitas yang berbeda.
Fungsi Linear
f(x)=2x+1

Pada fungsi linear, setiap perubahan input menghasilkan perubahan output yang tetap karena gradiennya konstan.

x f(x)
1 3
2 5
3 7
Setiap kenaikan 1 pada x menyebabkan nilai f(x) bertambah 2.
Fungsi Kuadrat
f(x)=x²

Pada fungsi kuadrat, perubahan output tidak tetap. Semakin besar nilai x, perubahan output akan semakin cepat.

x f(x)
1 1
2 4
3 9
Semakin besar nilai x, semakin besar pula perubahan nilai f(x).
Perbandingan Sensitivitas
Aspek Fungsi Linear Fungsi Kuadrat
Gradien Tetap Berubah-ubah
Perubahan Output Konstan Semakin besar
Grafik Garis lurus Kurva parabola
Sensitivitas Stabil Meningkat seiring bertambahnya x
Kesimpulan: Fungsi linear memiliki sensitivitas yang tetap karena gradiennya konstan, sedangkan fungsi kuadrat memiliki sensitivitas yang berubah sehingga perubahan output menjadi semakin besar ketika nilai input meningkat.

1.9 Analisis Numerik Sederhana

Analisis numerik sederhana digunakan untuk memahami perilaku suatu fungsi melalui perhitungan nilai fungsi pada beberapa input sehingga dapat diketahui pola perubahan yang terjadi.
Studi Kasus

Diketahui fungsi linear:

f(x)=2x+3

Hitung nilai fungsi untuk beberapa nilai x.

x 0 1 2 3 4
f(x) 3 5 7 9 11

Dari tabel di atas terlihat bahwa setiap kenaikan nilai x sebesar 1 menyebabkan nilai f(x) bertambah 2. Hal ini menunjukkan bahwa fungsi linear memiliki laju perubahan yang konstan.

Langkah Analisis
  1. Menentukan bentuk fungsi yang akan dianalisis.
  2. Menentukan beberapa nilai input (x).
  3. Menghitung nilai fungsi untuk setiap input.
  4. Membandingkan perubahan nilai output.
  5. Menarik kesimpulan berdasarkan hasil perhitungan.
Hasil Analisis
  • Nilai fungsi meningkat secara teratur.
  • Laju perubahan bernilai tetap sebesar 2.
  • Grafik fungsi berbentuk garis lurus.
  • Semakin besar nilai input, nilai output juga semakin besar.
Kesimpulan: Analisis numerik membantu memahami karakteristik suatu fungsi melalui data perhitungan. Dengan menghitung beberapa nilai input dan output, kita dapat mengetahui pola perubahan, laju perubahan, serta sifat dari fungsi tersebut.

Contoh Soal

Berikut beberapa contoh soal beserta penyelesaiannya untuk membantu memahami materi fungsi.
Contoh Soal 1

Soal:

Diketahui fungsi f(x)=2x+5. Tentukan domain, kodomain, range, dan nilai f(4).

Penyelesaian:

  • Domain = ℝ
  • Kodomain = ℝ
  • Range = ℝ
  • f(4)=2(4)+5=13
Fungsi tersebut merupakan fungsi linear karena berbentuk ax+b.
Contoh Soal 2

Soal:

Diketahui:

  • f(x)=2x+1
  • g(x)=x²

Tentukan hasil komposisi (f∘g)(x).

Penyelesaian:

Diketahui:

(f∘g)(x)=f(g(x)) =2(x²)+1 =2x²+1
Jadi, hasil komposisi fungsi (f∘g)(x)=2x²+1.

  1. Soal 1

    Diketahui fungsi f(x)=3x-2. Tentukan:

    1. Domain
    2. Kodomain
    3. Range
    4. Nilai f(5)
    Lihat Pembahasan

    Jawaban:

    • Domain = ℝ
    • Kodomain = ℝ
    • Range = ℝ
    • f(5)=3(5)-2=13

    Karena fungsi berbentuk ax+b, maka fungsi ini merupakan fungsi linear sehingga domain, kodomain, dan range-nya adalah himpunan bilangan real.

  2. Soal 2

    Diketahui:

    • f(x)=2x+1
    • g(x)=x²

    Tentukan hasil komposisi (f∘g)(x).

    Lihat Pembahasan
    (f∘g)(x)=f(g(x)) =2(x²)+1 =2x²+1

    Pada komposisi fungsi, hasil dari fungsi g(x) dimasukkan sebagai input ke fungsi f(x).

  3. Soal 3

    Diketahui fungsi f(x)=2x+1. Tentukan laju perubahan fungsi pada interval x=2 sampai x=5.

    Lihat Pembahasan

    Langkah 1

    • f(2)=5
    • f(5)=11

    Langkah 2

    Laju Perubahan = (11−5)/(5−2) =6/3=2

    Nilai 2 menunjukkan bahwa setiap kenaikan 1 satuan pada input akan menyebabkan output bertambah 2 satuan.
