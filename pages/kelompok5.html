Pendekatan Numerik Dan Turunan

Kelompok 5 Penjelasan

Pendekatan numerik adalah metode untuk menyelesaikan masalah matematika menggunakan perhitungan angka secara bertahap (aproksimasi) ketika solusi eksak sulit atau tidak dapat diperoleh secara langsung.

    • 5.1 Pendekatan Numerik Nilai Approx

    Apa Itu Pendekatan Numerik?
    Pendekatan numerik adalah cara menghitung suatu nilai (misalnya turunan) dengan menggunakan angka-angka pendekatan, bukan hasil rumus yang pasti. Disini terdapat 3 cara mementukan nilai approx/aproksimasi yaitu dengan Forward Difference, Backward Difference, dan Central Difference

    1. Metode Fordward : $$f'(x) \approx \frac{f(x + h) - f(x)}{h}$$

    Keterangan

    f'(x) = Turunan Pertama Fungsi f terhadap x
    f(x) = Nilai fungsi f di titik x
    x = Titik di mana turunan dihitung
    h = Perubahan kecil pada x (misalnya 0.001)
    f(x + h) = Nilai fungsi f di titik x + h

    Contoh Soal Latihan Forward Difference

    Kerjakanlah Contoh Soal Dengan
    X³ || X = 1 || H = 0.2
    Cara Pengerjaan
    f(1) = 1³ = "1"
    f (x + h) = 1 + 2 = 1,2
    f(1,2) = 1,2³ = "1,728"
    $$f'(x) \approx \frac{f(1,728) - f(1)}{0,2}$$ $$f'(x) = 3,64 $$



    2. Metode Backward : $$f'(x) \approx \frac{f(x) - f(x - h)}{h}$$

    Keterangan

    f'(x) = Turunan Pertama Fungsi f terhadap x
    f(x) = Nilai fungsi f di titik x
    x = Titik di mana turunan dihitung
    h = Perubahan kecil pada x (misalnya 0.001)
    f(x - h) = Nilai fungsi f di titik x - h

    Contol Soal Latihan Backward Difference

    X⁴ || X = 3 || H = 0.1
    Cara Pengerjaan
    f(1) = 3⁴ = "81"
    f (x - h) = 3 - 0,1 = 2,9
    f(2,9) = 2,9⁴ = "70,7281"
    $$f'(x) \approx \frac{f(81) - f(70,7281)}{0,1}$$ $$f'(x) = 102,719 $$



    3. Metode Central : $$f'(x) \approx \frac{f(x + h) - f(x - h)}{2h}$$

    Keterangan

    f'(x) = Turunan Pertama Fungsi f terhadap x
    x = Titik di mana turunan dihitung
    h = Perubahan kecil pada x (misalnya 0.001)
    f(x + h) = Nilai fungsi f di titik x + h
    f(x - h) = Nilai fungsi f di titik x - h

    Contoh Soal Latihan Central Difference

    X³ + 2X || X = 2 || H = 0.1
    Cara Pengerjaan
    f (x + h) = 2 + 0,1 = 2,1
    f(2,1) = 2,1³ + 2 x 2,1 = "13,461"

    f (x - h) = 2 - 0,1 = 1,9
    f(1,9) = 1,9³ + 2 x 1,9 = "10,659"
    $$f'(x) \approx \frac{f(13,461) - f(10,659)}{2 x 0,1}$$ $$f'(x) \approx \frac{2,802}{2 x 0,1}$$ $$ \approx 14,01 $$



    5.2 Pendekatan Numerik Nilai True dan Galat (Error)

    Area Materi — Kelompok 5
    Metode analitik adalah cara menyelesaikan masalah matematika menggunakan rumus dan aturan matematika secara langsung untuk mendapatkan hasil yang tepat (eksak) tanpa perkiraan.

    Galat (error) adalah selisih antara nilai eksak (true value) dengan nilai pendekatan (approximation)

    Rumus Numerik Nilai True $$x^n \rightarrow n x^{n-1}$$

    Keterangan

    x = Titik di mana turunan dihitung
    n = Bilangan Pangkat

    Rumus Error (Galat) $$\text{Error} = |x_{\text{true}} - x_{\text{approx}}|$$

    Keterangan

    $$\text{Error}$$ = Nilai galat/kesalahan mutlak (selisih murni)
    $$x_{\text{true}} $$ = Nilai Sebenarnya
    $$x_{\text{approx}}$$ = Nilai Aproksimasi

    5.3 Kriteria metode numerik yang baik

    Area Materi — Kelompok 5
    Kriteria metode numerik yang baik adalah ukuran untuk menilai apakah suatu metode pendekatan dalam perhitungan sudah layak digunakan atau tidak. suatu metode dikatakan baik jika memenuhi tiga hal utama yaitu akurat, stabil, dan efisien.

    Akurat $$\text{Error} = |x_{\text{true}} - x_{\text{approx}}|$$

    Akurat adalah ukuran yang menunjukkan “Seberapa dekat hasil perhitungan dengan nilai sebenarnya”. Semakin kecil selisih (Error) antara hasil pendekatan dan nilai asli, maka hasil tersebut semakin akurat.

    Keterangan Komponen:

    Error = Nilai galat mutlak (selisih murni antara nilai asli dan pendekatan)
    xapprox = Nilai hasil pendekatan / hasil perhitungan metode numerik
    xtrue = Nilai asli / eksak dari perhitungan matematika murni

    Uji Turunan Pertama

    [Kelompok 5: jelaskan uji turunan pertama untuk menentukan titik ekstrem]



    Stabil $$h \downarrow \implies \text{hasil mendekati nilai asli secara konsisten}$$

    Stabil adalah ukuran yang menunjukkan seberapa konsisten hasil perhitungan ketika terjadi perubahan kecil pada input atau langkah (seperti nilai h). Jika perubahan kecil tidak menyebabkan hasil berubah drastis, maka metode tersebut dikatakan stabil.

    Kesimpulan Logika:

    Perubahan kecil input → Perubahan kecil hasil
    Nilai h diperkecil (h ↓) ⇒ Hasil mendekati nilai asli secara konsisten

    Uji Turunan Pertama

    [Kelompok 5: jelaskan uji turunan pertama untuk menentukan titik ekstrem]



    Efisiensi $$\text{Efisiensi} = \frac{\text{hasil (akurasi)}}{\text{waktu / langkah}}$$

    Efisien adalah ukuran yang menunjukkan seberapa hemat suatu metode dalam penggunaan waktu dan jumlah perhitungan. Metode dikatakan efisien jika dapat memberikan hasil yang cukup baik dengan langkah yang lebih sedikit atau proses yang lebih cepat.

    Secara Konsep Komputasi:

    Langkah Perhitungan = Sedikit / minimal
    Waktu Eksekusi = Cepat (Running time singkat)
    Hasil Pendekatan = Cukup baik dan memenuhi toleransi

    5.4 Interpretasi hasil turunan sebagai laju perubahan

    Area Materi — Kelompok 5
    Interpretasi hasil turunan sebagai laju perubahan adalah cara memahami nilai turunan sebagai ukuran seberapa cepat suatu besaran berubah terhadap variabelnya. Nilai turunan menunjukkan perubahan kecil pada nilai fungsi ketika variabel inputnya berubah sedikit, sehingga dapat digunakan untuk menggambarkan kecepatan perubahan suatu fenomena, seperti kecepatan, pertumbuhan, atau penurunan dalam berbagai konteks.

    A. Tiga Skema Rumus Pendekatan Numerik:
    $$\text{1. Beda Maju (Forward): } f'(x) \approx \frac{f(x + h) - f(x)}{h}$$ $$\text{2. Beda Mundur (Backward): } f'(x) \approx \frac{f(x) - f(x - h)}{h}$$ $$\text{3. Beda Pusat (Central): } f'(x) \approx \frac{f(x + h) - f(x - h)}{2h}$$
    Pengertian Variabel Simbol:

    f'(x) ≈ Nilai laju perubahan sesaat (turunan pertama) hasil pendekatan komputer.
    x = Titik evaluasi utama atau posisi kondisi saat ini.
    h = Ukuran langkah (step size) atau interval perubahan kecil pada input.

    B. Panduan Universal Interpretasi Hasil:
    Untuk mengartikan angka hasil hitungan: $$f'(x) = L$$ gunakan kalimat baku di bawah ini:
    "Pada saat variabel input berada di titik (x), maka setiap kenaikan 1 satuan pada input akan menyebabkan variabel output (f(x)) bertambah/berubah sebesar (L) satuan."



    5.5 Penerapan Turunan Numerik Dalam Bidang IT

    Area Materi — Kelompok 5
    Di bidang IT, turunan numerik sangat penting karena komputer tidak dapat memproses persamaan kalkulus analitis yang kompleks secara langsung, melainkan harus menggunakan pendekatan algoritma aritmatika untuk menghitung laju perubahan data secara real-time. Konsep ini menjadi fondasi utama dalam kecerdasan buatan (Machine Learning) untuk memperbarui bobot algoritma lewat perhitungan gradien error, dalam pengembangan game untuk memproses pergerakan animasi serta simulasi fisika yang mulus, hingga dalam pengolahan citra digital untuk mendeteksi tepi objek berdasarkan perubahan intensitas warna pixel. Melalui pendekatan numerik, sistem komputasi mampu melakukan jutaan kalkulasi perubahan data yang masif secara akurat, stabil, dan efisien hanya dalam hitungan milidetik.

    Penerapan Turunan dalam Bidang IT: Pentingnya Turunan Numerik Di Bidang IT

    Turunan numerik memegang peranan krusial dalam pengembangan perangkat lunak dan teknologi informasi melalui tiga pilar kriteria utama:

    🎯 Akurat Turunan numerik memungkinkan perangkat lunak untuk mencapai tingkat presisi yang tinggi. Turunan juga digunakan untuk menghitung error skala terkecil hingga terbesar.
    ⚓ Stabil Sifat stabil ini sering kali dipakai dalam Simulasi Fisika Game, atau pun animasi seperti karakter melakukan gerakan yang cepat. Perhitungan kecepatannya tetap sehingga tidak terlihat patah-patah.
    Logika fisis: $$h \downarrow \implies \text{hasil mendekati nilai asli secara konsisten}$$ (perubahan kecil input → perubahan kecil hasil).
    ⚡ Efisien Turunan numerik dirancangkan agar dapat diproses oleh komputer dengan sangat cepat. Dengan Turunan Numerik ini juga komputer bisa melakukan jutaan perhitungan perubahan dalam hitungan milidetik.
