7.1 Menentukan titik maksimum dan minimum

Materi — Kelompok 7
Di Dalam kalkulus, turunan digunakan untuk melihat perubahan suatu fungsi dimana suatu fungsi kita dapat menentukan
Titik Maksimum -> nilai tertinggi
Titik Minimum -> Nilai terendahnya

[Contoh Soal]


f(x) = x² -4x + 3 ,
Tentukan titik minimum
dari fungsi ini

[penjelasan Soal]

[Langkah Pertama: Turunkan fungsi]
f(x) = x² - 4x + 3
f'(x) = 2x + 4
Lalu Cari Titik kritisnya
f(x) = 2x + 4 = 0
x = 2
f"(x) = 2

[Langkah Kedua:Lalu ubah x = 2]
f(x) = x² -4x + 3
f(2)= (2)² - 4(2) +3
= 4 - 8 + 3 = -1
Jadi Titik minimun dari f(x) = x² -4x - 3 adalah (2, -1)

— Teorema

7.2 Menentukan titik stasioner

Materi — Kelompok 7
Syarat Untuk Menghitung Titik Stasioner adalah dengan turunan pertama sama dengan nol
Titik stasioner juga dapat berupa maksimum lokal (puncak), minimum lokal (lembah), atau titik belok (inflection point) dimana fungsi tidak berubah arah.

[Kelompok 7: isi penjelasan langkah-langkah volume benda putar]

Contoh Soal

Contoh Soal — Titik Stasioner

Soal:

[Diketahui fungsi :
f(x) = x² − 4x + 3
Tentukan titik stasionernya.]

Penyelesaian:

[Turunkan fungsi:
f′(x) = 2x − 4]
Cari nilai saat turunan sama dengan nol:
2x − 4 = 0
x = 2]
Cari nilai fungsi
f(2) = 2² − 4(2) + 3
= 4 − 8 + 3
= −1
Jadi titik stasionernya adalah:
(2, −1)

7.3 Penerapan optimasi

Materi — Kelompok 7
Optimasi dalam Kalkulus adalah metode untuk mencari nilai maksimum atau minimum dari suatu fungsi menggunakan turunan.
Langkah Umum Optimasi
Tentukan fungsi (model matematika)
Turunkan fungsi →
Cari titik stasioner →
Uji dengan turunan kedua Interpretasi hasil

[Kelompok 7: isi penjelasan langkah-langkah volume benda putar]

Contoh Soal

Contoh Soal

Soal:

[Diketahui fungsi :
P(x)=−x^2+20x+5]

Penyelesaian:

Langkah 1:
Menentukan Turunan Pertama
Turunkan fungsi keuntungan terhadap x:
P′(x)=−2x+20
Langkah 2:
menentukan titik kritis
Set P′(x) = 0:
−2x + 20 = 0
x = 10
Dalam optimasi, nilai maksimum atau minimum terjadi saat:
P′(x)=0P'(x)=0P′(x)=0
Maka:
−2x+20=0 -> Pindahkan 20 ke ruas kanan:
−2x=−20 -> Bagi kedua ruas dengan -2:
x=10

Langkah 3:
Menghitung Keuntungan Maksimum
Substitusikan x=10 ke fungsi awal:
P(10)=−(10)^2+20(10)+5
P(10)=−100+200+5
P(10)=105
